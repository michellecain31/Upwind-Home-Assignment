# PenguWave: Security Operations Portal

A security operations portal for monitoring security events across your infrastructure.

## Overview

This is a **frontend-only** starter application with mock data and an API contract. It's the starting point for a home assignment where you'll turn it into a working full-stack application.

Full task details, scope, and submission guidelines are provided in the accompanying assignment PDF.

## Prerequisites

- Node.js 18+
- Any backend language, framework, and database of your choice

## Getting Started

```bash
npm install
npm run dev
```

The frontend runs at http://localhost:5173.

You'll see a login modal, an events table with 50 mock security events, and a users management page. The app works standalone with mock data, so feel free to explore it before you start building.

## What's Included

- React + Vite + TypeScript frontend (3 pages)
- 50 realistic mock security events (`data/mock_events.json`)
- API endpoint contract (`docs/api_contract.md`)
- No backend (that's for you to build)
